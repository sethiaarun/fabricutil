# fabricutil
Fabric Utility Programs

## Tools

1. **Livy API Example** - [livyapiexample](Livy/livyapiexample.py) - [Blog](https://medium.com/@sethia.arun/introduction-017b46ff4bef)

2. **ADO Test Results Analyzer** - [ado-test-results-analyzer](ado-test-results-analyzer/) - Analyze JUnit XML test results from Azure DevOps (ADO) pipelines
   - `ado_test_results_analyzer.py` - Parse ADO pipeline test result zip files and generate CSV/HTML failure reports
   - `ado_test_results_compare.py` - Compare two ADO pipeline test runs to identify regressions and fixes
